# Vogent Documentation

Unofficial mirror of [Vogent](https://vogent.ai) documentation for [Context7](https://context7.com) integration.

## About Vogent

Vogent is an all-in-one platform for building humanlike, intelligent voice AI agents. Features include:

- **IVR Detection & Navigation** - Automatically detect and navigate phone trees
- **Custom Voice Models** - Clone voices or use Sesame CSM-1B ultra-low-latency TTS
- **Flow Builder** - Visual interface for structured conversation paths
- **Webhook Integrations** - Real-time notifications for dial events, transcripts, and function calls
- **Batch Dial Jobs** - Scale outbound campaigns with queued dials
- **HIPAA-Compliant Workspaces** - For healthcare applications
