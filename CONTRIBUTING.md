# Contributing to Vogent Documentation

Thank you for your interest in improving this documentation mirror!

## How to Contribute

### Reporting Issues

- **Outdated content**: If documentation is out of sync with [docs.vogent.ai](https://docs.vogent.ai), open an issue
- **Broken links**: Report any dead links or missing pages
- **Typos/errors**: Small fixes can go directly to a PR

### Making Changes

1. **Fork** this repository
2. **Create a branch** for your changes (`git checkout -b fix/typo-in-quickstart`)
3. **Make your changes** in the `docs/` directory
4. **Test locally** if possible
5. **Submit a PR** with a clear description

### Documentation Standards

- Keep markdown formatting consistent with existing files
- Preserve the file naming convention: `category_subcategory_page-name.md`
- Don't remove the YAML frontmatter if present
- Code examples should be complete and runnable

### What We Accept

- Fixes to typos, grammar, or formatting
- Clarifications to confusing documentation
- Additional code examples
- Improvements to Context7 integration

### What We Don't Accept

- Content not present in official Vogent docs
- Promotional or marketing content
- Changes that would break Context7 indexing

## Questions?

Open an issue or reach out on the [Vogent Discord](https://discord.gg/An5z6xhYfS).

This is a community project. All contributions are appreciated!
