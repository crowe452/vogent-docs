Getting Started

1

Create an account

Go to the [Vogent dashboard](https://app.vogent.ai) and sign in with a Google or Microsoft account.

2

Create a phone number

Click on **Call Settings** on the left sidebar, then click **Add Phone Number**.

![](https://mintcdn.com/elto-1/ooEImm5m-H0BmxUg/images/add-number-light.png?fit=max&auto=format&n=ooEImm5m-H0BmxUg&q=85&s=c6aef1867566bf8590a99344de5703ad)![](https://mintcdn.com/elto-1/ooEImm5m-H0BmxUg/images/add-number-dark.png?fit=max&auto=format&n=ooEImm5m-H0BmxUg&q=85&s=58e856304d6394c95a51c7dd75d903a9)

Then enter a 3-digit area code and click the magnifying glass icon to search for a phone number.

![](https://mintcdn.com/elto-1/ooEImm5m-H0BmxUg/images/area-code-light.png?fit=max&auto=format&n=ooEImm5m-H0BmxUg&q=85&s=6c9110a80e6369a8994c9de8f81e249e)![](https://mintcdn.com/elto-1/ooEImm5m-H0BmxUg/images/area-code-dark.png?fit=max&auto=format&n=ooEImm5m-H0BmxUg&q=85&s=3df88db6a1c0fceed72abcc607b695cf)

Then click **Add** to add the phone number to your workspace. 

Availability varies by area code. `510` is a popular area code and is generally available, so it’s a good one to use for this demo.
