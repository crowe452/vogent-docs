Get Started

Welcome to the Vogent API docs. The root of the Vogent API is at `https://api.vogent.ai/api`.

## 

[​](#authorization)

Authorization

You can authenticate with the Vogent API with an `Authorization` header in any request you send to the API. You can generate an API key on your Vogent dashboard at `https://app.vogent.ai` by going to the API Keys page in the sidebar.
